# AKPsi Project F1 Site

An interactive Formula 1 educational website built with React and Three.js.

## Features
- 🚗 Interactive 3D F1 car with hoverable part annotations
- 👨‍🏎️ Team & driver profiles for Red Bull, Ferrari, and McLaren
- 📊 Car specifications and team comparison stats

## Tech Stack
- React
- Three.js (3D rendering)
- Base44 (hosting)
